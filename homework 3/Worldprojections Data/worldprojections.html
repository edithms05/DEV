In this project I learned how to display images in different projections

I first downloaded the Indicitrix mapper program so that it was easy to clearly identify where warping on different projections was occuring. From here, I was able to change the Project Coordinate Reference System a number of times to show what different projections look like with the Indicatrix. Each time I would use a new projection, I would create a new Print Layout so that I could download each as a PNG and put it into GitHub.

WGS84 Projection

In this projection, the warping is present the further one gets from the equator. This is represented by the Indicatrix.

Aitoff Projection

This projection is represented more spherically which allows for less warping along the poles. There is more warping when you move East or West, however.

Winkel Projection

The Winkle projection is very similar to the Aitoff projection in that it shows the Earth as an ellipsoid. There is slightly less warping to the East and West, however.

Psuedo Mercator

This projection is also very common. There is little to no warping around the equator, but there is quitea bit around the poles. A good example of this is how large Greenland is represented in this map. It appears to be as large as Africa which is not accurate.

Equal Area Cyclical Projection

As the name indicates, this projection looks like a flattened cylinder. This allows for minimal warping across the equator, but a good deal of warping around the poles.

Equidistant Conic Projection

This equidistant conic projection is viewd from the North Pole. This allows for minimal warping around the Northern Pole, unlike the projections above, but creates extreme warping around the South Pole.

Azimuthal Equidistant Projection

This prokection is very similar to the equidistant conic projection, but rather than stopping part way, the projection wraps around the entirety of the globe.

Robinson Projection

This projection is sort of a cross between the Aitoff or Winkel Projections and an Equal Area Cyclical Projection.

Bonne Projection

The Bonne Projection is interesting in that there is warping in many different areas of the globe, but it provides for less warping around the North Pole.
