In this project I learned how to display images in different projections

Describe in your own words how you displayed the map in different projections using QGIS

In QGIS, I was able to display the different projection styles by changing the Project Coordinate Reference System. From here, downloading the Indicatrix mapper allowed me to look at the immediate visual differences in each projection, while also looking at how and where each one was warped. To creat a png for each projection to better showcase these differences, I used Project Layout, and then added labels to make sure each was easy to distinguish, especially because some of them look so similar.

WGS84 Projection

This projection is fairly common, and one of the first options that you see in QGIS. It is very obviously warped the closer that you get to the poles, while it stays more accurate closer to the poles.

Aitoff Projection

This projection is more spherical, and shows less warping near the poles. In this case, the projection represents the Earth's shape as an ellipsoid.

Psuedo Mercator Projection

The Psuedo Mercator Projection is very similar the WGS84 projection in that they both show the world in a way that creates warping near the poles. In this projection, however, it is not warped as horizontally as the WGS84 projection is.

Winkel Projection

The Winkel Projection is similar to the Aitoff Projection in that it is representing the Earth as more of an ellipsoid, but this projection has less dramatic warping as you move away from the equator, and further east/west.
